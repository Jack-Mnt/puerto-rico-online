# Skill: Branding PuertoRico La Licorería

Usa este skill cuando trabajes en componentes visuales, estilos, interfaz, tarjetas, banners, navegación, botones, formularios o cualquier ajuste relacionado con la identidad visual de PuertoRico La Licorería.

## Contexto de marca

PuertoRico La Licorería es una licorería familiar ubicada en Ica, Perú. El negocio fue fundado por mi familia hace más de 30 años y ahora buscamos modernizarlo con una tienda online.

La web debe verse moderna, premium, elegante y confiable, pero también cercana y fácil de usar.

La marca debe transmitir:

* Elegancia.
* Confianza.
* Cercanía.
* Producto original.
* Experiencia premium.
* Compra fácil y rápida.

## Paleta de colores

Usar como dirección visual:

* Negro profundo / carbón: color principal para fondos premium, header, footer o secciones elegantes.
* Dorado champagne / crema: color principal de detalle para botones, acentos, bordes, títulos destacados o elementos premium.
* Cobre / whisky ámbar: color cálido de acento para hover, etiquetas, detalles secundarios o llamadas de atención.
* Blanco / crema suave: color para tarjetas de productos, fondos limpios y zonas donde se muestran imágenes de productos.

Las tarjetas de productos deben mantenerse claras, preferiblemente blancas o crema suave, porque muchas imágenes de productos tienen fondo blanco.

Evita que toda la web sea oscura. La web debe sentirse premium, pero también clara y fácil de comprar.

## Tipografías

Usar:

* Michroma para logo, títulos principales, headings destacados o elementos de marca.
* Inter para textos generales, interfaz, botones, formularios, descripciones, precios y navegación.

No usar demasiadas tipografías. Mantener consistencia visual.

## Componentes

### Tarjetas de producto

Las tarjetas deben ser:

* Claras.
* Limpias.
* Con buen espaciado.
* Con borde sutil o sombra ligera.
* Con precio visible.
* Con botón claro de acción.
* Con imagen protagonista.

Evitar fondos oscuros dentro de la tarjeta de producto.

### Botones

Los botones principales deben destacar usando dorado champagne, crema o cobre/ámbar, según el contexto.

Los botones deben sentirse premium, no chillones.

Evitar colores saturados sin relación con la marca.

### Header y navegación

El header puede usar negro carbón o fondo claro según la sección, pero debe sentirse elegante y ordenado.

El carrito debe ser visible y fácil de encontrar, especialmente en mobile.

### Banners

Los banners deben ser premium, limpios y no saturados.

Evitar exceso de luces, brillos fuertes o efectos recargados.

Preferir composición minimalista, buen contraste y textos cortos.

## Responsive

La web debe estar optimizada para mobile.

Priorizar:

* Lectura rápida.
* Botones grandes y fáciles de tocar.
* Carrito accesible.
* Filtros simples.
* Imágenes optimizadas.
* Evitar saturación visual.

## Regla general

Antes de modificar estilos, revisa cómo está implementado actualmente el diseño. Mantén compatibilidad con la estructura existente, Tailwind o sistema de estilos usado en el proyecto.

No cambies toda la identidad visual sin una razón clara. Haz ajustes incrementales, limpios y coherentes.
