# Skill: Componentes UI PuertoRico Online

Usa este skill cuando trabajes en componentes visuales, tarjetas, banners, filtros, formularios, botones, checkout, carrito, navegación o responsive.

## Principios UI

La interfaz debe ser:

* Clara.
* Rápida.
* Premium.
* Fácil de usar en mobile.
* Adaptada a clientes reales de una licorería.
* Sin exceso de pasos.
* Sin textos confusos.

## Mobile first

Prioriza mobile.

Muchos clientes usarán la tienda desde celular.

En mobile:

* Botones grandes.
* Carrito visible.
* Filtros fáciles.
* Texto legible.
* Checkout simple.
* Imágenes optimizadas.
* Evitar pantallas saturadas.

## Tarjetas de producto

Las tarjetas deben ser claras porque las imágenes de productos suelen tener fondo blanco.

Recomendaciones:

* Fondo blanco o crema suave.
* Borde sutil.
* Sombra ligera si aporta profundidad.
* Imagen bien centrada.
* Nombre legible.
* Marca o categoría si aporta valor.
* Precio visible.
* Botón de acción claro.
* Espaciado suficiente.

## Filtros

Los filtros deben ayudar, no complicar.

Categorías y marcas deben ser fáciles de seleccionar.

Si el usuario elige una categoría, las marcas deberían filtrarse según esa categoría cuando la lógica lo permita.

Evitar listas demasiado largas sin orden visual.

## Botones

Los botones deben tener jerarquía:

* Primario: acción principal.
* Secundario: acción alternativa.
* Peligro: cancelar, eliminar o rechazar.

No usar demasiados estilos de botón diferentes.

## Textos

Usa textos simples y entendibles.

No uses “Pick up” como texto visible.

Usa:

* Recojo en tienda

## Formularios

Los formularios deben pedir solo lo necesario.

La dirección puede ser opcional.

No compliques el checkout con campos innecesarios.

## Banners

Los banners deben ser:

* Limpios.
* Premium.
* Poco saturados.
* Con texto corto.
* Con buen contraste.
* Responsivos.

Evitar efectos excesivos, brillos fuertes o demasiados elementos visuales.

## Regla general

Antes de crear componentes nuevos, revisa si ya existe un componente similar.

Evita duplicar lógica o estilos.

Mantén consistencia visual en toda la web.
